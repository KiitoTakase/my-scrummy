# テンション爆上げライト

## これは何？
気分が落ち込んでいたり、元気が出ないときに、
部屋を明るくして気持ちまで少し前向きにしてくれるIoTデバイスです。

## 動作内容
- 光センサーで部屋の明るさを測る
- 部屋が暗いと判断したらRGB LEDが点灯する
- LEDが虹色に光り、空間を明るく楽しい雰囲気にする
- 部屋が明るくなるとライトは自動で消える

## 工夫したポイント
ただ部屋を明るくするだけではなく、
「気持ちまで少し明るくすること」をテーマに企画しました。
虹色に変化するライトで、見ているだけでも少しテンションが上がるように工夫しました。

## 使用した部品
- 光センサー
- RGB LED
- マイコン
